# Rena.edu

A content-forward marketing and booking request website built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/` - App Router directory containing pages and layouts
- `src/app/layout.tsx` - Root layout component
- `src/app/page.tsx` - Home page
- `src/app/globals.css` - Global styles with Tailwind CSS

## Features

- Next.js 14 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- Responsive design
- Booking request form
- Marketing content sections

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```
